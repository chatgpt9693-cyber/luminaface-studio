# 🤖 Настройка Telegram Mini App

Пошаговая инструкция по созданию Telegram Mini App для Belka_faces.

## Шаг 1: Создание бота через BotFather

1. Открой Telegram и найди [@BotFather](https://t.me/BotFather)
2. Отправь команду `/newbot`
3. Введи название бота: `Belka Faces`
4. Введи username бота: `belka_faces_bot` (или любой доступный)
5. Сохрани токен бота (понадобится позже)

## Шаг 2: Настройка Web App

1. Отправь BotFather команду `/newapp`
2. Выбери своего бота из списка
3. Введи название приложения: `Belka Faces`
4. Введи описание:
   ```
   Премиум массаж лица — запись онлайн. Управление студией и бронирование процедур.
   ```
5. Загрузи фото (квадратное, минимум 640x640px)
6. Загрузи GIF-демо (опционально)
7. Введи URL приложения: `https://zapisi.netlify.app`
8. Введи короткое название: `belkafaces`

## Шаг 3: Настройка меню бота

1. Отправь BotFather команду `/setmenubutton`
2. Выбери своего бота
3. Введи текст кнопки: `🌸 Открыть приложение`
4. Введи URL: `https://zapisi.netlify.app`

## Шаг 4: Настройка команд бота

1. Отправь BotFather команду `/setcommands`
2. Выбери своего бота
3. Введи список команд:
   ```
   start - Запустить приложение
   booking - Записаться на процедуру
   history - История визитов
   help - Помощь
   ```

## Шаг 5: Настройка описания

1. Отправь BotFather команду `/setdescription`
2. Выбери своего бота
3. Введи описание:
   ```
   Добро пожаловать в Belka Faces! 🌸
   
   Премиум массаж лица в Минске.
   
   ✨ Классический массаж лица
   ✨ Массаж лица Асахи
   ✨ Буккальный массаж лица
   
   Запишитесь на процедуру прямо в приложении!
   ```

## Шаг 6: Тестирование

1. Найди своего бота в Telegram
2. Нажми `/start`
3. Нажми кнопку "🌸 Открыть приложение"
4. Приложение должно открыться внутри Telegram

## Дополнительные возможности

### Автоматический вход через Telegram

Приложение автоматически определяет, что открыто в Telegram, и может использовать данные пользователя:
- ID пользователя
- Имя и фамилия
- Username
- Язык интерфейса

### Haptic Feedback (вибрация)

Приложение использует тактильную обратную связь при нажатиях на кнопки.

### Кнопка "Назад"

Автоматически показывается при навигации по приложению.

### Главная кнопка

Используется для подтверждения действий (например, "Подтвердить запись").

## Настройка уведомлений (опционально)

Для отправки уведомлений о записях нужно:

1. Сохранить токен бота в переменные окружения:
   ```
   TELEGRAM_BOT_TOKEN=твой_токен_от_BotFather
   ```

2. При регистрации/входе сохранять Telegram ID пользователя в базу данных

3. Настроить отправку уведомлений через Telegram Bot API

## Публикация в каталоге Telegram

После тестирования можно опубликовать бота:

1. Отправь BotFather команду `/setabouttext`
2. Введи краткое описание (до 120 символов)
3. Бот появится в поиске Telegram

## Troubleshooting

### Приложение не открывается
- Проверь, что URL правильный: `https://zapisi.netlify.app`
- Убедись, что сайт доступен по HTTPS
- Проверь, что переменные окружения настроены на Netlify

### Не работает авторизация
- Убедись, что переменные `VITE_SUPABASE_URL` и `VITE_SUPABASE_ANON_KEY` настроены на Netlify
- Проверь консоль браузера на ошибки

### Не приходят уведомления
- Проверь, что токен бота правильный
- Убедись, что пользователь нажал `/start` в боте
- Проверь, что Telegram ID сохраняется в базе данных
