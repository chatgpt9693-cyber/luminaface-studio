# Belka_faces - Премиум массаж лица

Система управления студией массажа лица с функциями для мастера и клиентов.

## 🚀 Что реализовано

### ✅ Авторизация (100%)
- Вход в систему с ролями (Мастер / Клиент)
- Защищённые роуты
- Демо-аккаунты:
  - Мастер: `master@belkafaces.ru` / `demo123`
  - Клиент: `client@example.ru` / `demo123`

### ✅ Функционал мастера (100%)
- **Дашборд**: статистика, график доходов, расписание на сегодня
- **Календарь**: создание/редактирование записей, интерактивная сетка
- **Клиенты**: CRUD операции, поиск по имени/телефону/email
- **Услуги**: CRUD операции, управление ценами и длительностью
- **Доходы**: графики по месяцам и услугам, экспорт в CSV

### ✅ Функционал клиента (100%)
- **Дашборд**: приветствие, ближайшая запись, статистика
- **Бронирование**: 3-шаговый процесс (услуга → время → подтверждение)
- **История**: список процедур с визуализацией зон на 3D-модели лица
- **3D-модель**: интерактивные зоны с историей обработки

### ✅ UI/UX (100%)
- Тёмная тема с градиентами
- Анимации Framer Motion
- Адаптивный дизайн
- Toast-уведомления
- Модальные окна

## 🛠 Технологии

- React 18 + TypeScript
- Vite
- React Router v6
- TanStack Query
- Shadcn/ui + Radix UI
- Tailwind CSS
- Framer Motion
- Recharts

## 📦 Установка и запуск

### 1. Установка зависимостей
```bash
npm install
```

### 2. Настройка Supabase (опционально)

Если хочешь использовать реальную БД вместо моковых данных:

1. Открой Supabase SQL Editor
2. Скопируй и выполни скрипт из `supabase/schema.sql`
3. Создай `.env` файл (скопируй из `.env.example`)
4. Заполни свои ключи из Supabase (Project Settings → API)

### 3. Запуск

```bash
# Запуск dev-сервера
npm run dev

# Сборка для продакшена
npm run build

# Предпросмотр сборки
npm run preview
```
